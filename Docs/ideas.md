# Plot

## World

Setting is middle-age with high level of magic.

## Character

Character is a crime investigator in a world, where magic is common and so is crime.

## Main Story

Game will start with some sort of training tutorial, where the most common side-effects of using magic to commit or hide crime will be introduced to the user. These may include for instance the side-effects that victim of memory manipulation goes through (nausea, sense of confusion, ...) or training to spot subtle clues in the environment (deformed plants or wildlife, accelerated aging of materials, ...).
